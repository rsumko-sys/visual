# OSINT Command Center — Design System

## 1. UX принципи
- **Користувач понад усе**: інтуїтивна навігація, правило 3 кліків
- **Mobile First**: адаптивність з мобільного (xs) до десктопа
- **Accessibility**: WCAG контраст ≥4.5:1, мінімум 44×44px для кнопок

## 2. Кольори (60-30-10)
| Роль | Колір | Використання |
|------|-------|--------------|
| **60%** | `#0a0e17` | Основний фон |
| **30%** | `#111827` | Поверхні (Paper, картки) |
| **10%** | `#00d4aa` | CTA, посилання, акценти |
| Secondary | `#6366f1` | Додатковий акцент |

## 3. Текст
- **Primary**: `#f9fafb` (не чистий білий)
- **Secondary**: `#9ca3af`
- **Читабельність**: max-width 65ch для body, line-height 1.5

## 4. Сітка
- **Baseline**: 8px (MUI spacing)
- **Container**: maxWidth lg (1280px) для контенту
- **Gutter**: 24px (spacing: 3)

## 5. Кнопки
- **Touch target**: min 44×44px
- **Ripple**: увімкнено для зворотного зв'язку

## 6. Семантичні кольори
- Success: `#10b981`
- Error: `#ef4444`
- Warning: `#f59e0b`
